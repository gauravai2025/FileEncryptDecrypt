# File EncryptDecrypt

A C++ project that demonstrates **parallel file encryption and decryption** using two different approaches:

- **Multiprocessing (Child Processes)**
- **Multithreading with Shared Memory**

The project is designed to explore how parallelism can improve the performance of cryptographic operations while handling multiple files efficiently.

---

## Features

- Encrypt and decrypt multiple files.
- Parallel execution for improved performance.
- Queue-based task management.
- Two independent implementations:
  - Process-based parallelism using `fork()`.
  - Thread-based parallelism using POSIX threads.
- Shared memory and semaphore synchronization (multithreading branch).

---

## Branches

### 1. `add/childProcessing`

This branch implements parallel encryption and decryption using **child processes**.

#### Key Features

- Process creation using `fork()`
- Independent task execution by child processes
- Queue-based task scheduling
- Concurrent processing of multiple files

#### Technologies Used

- C++
- POSIX Process APIs
- `fork()`
- Process Management

---

### 2. `add/multithreading`

This branch implements parallel encryption and decryption using **multithreading** and **shared memory**.

#### Key Features

- POSIX threads (`pthread`)
- Shared memory communication
- Semaphore-based synchronization
- Concurrent execution of encryption and decryption tasks

#### Technologies Used

- C++
- POSIX Threads (`pthread`)
- Shared Memory
- Semaphores

## Prerequisites

Before running the project, ensure you have:

- GCC/G++ (C++17 or later)
- Make
- Python 3
- Linux/WSL environment (required for POSIX APIs)

Verify installation:

```bash
g++ --version
make --version
python --version
```

## Installation

Clone the repository:

```bash
git clone https://github.com/gauravai2025/FileEncryptDecrypt.git
cd FileEncryptDecrypt
```

Switch to the desired branch:

```bash
git checkout add/childProcessing
```

or

```bash
git checkout add/multithreading
```

---

## Build Instructions

### Create Test Directories

Generate sample directories and files:

```bash
python makeDirs.py
```

### Compile the Project

```bash
make
```

### Run the Application

```bash
./encrypty
```

---

## Usage

After starting the application:

1. Enter the directory name generated by `makeDirs.py`

Example:

```text
test
```

2. Choose an operation:

```text
ENCRYPT
```

or

```text
DECRYPT
```

Example:

```text
Enter directory name:
test

Enter operation:
ENCRYPT
```

The application will process the files using the selected parallelization technique.
